## 🔧 技术细节

### 文件结构
```
workspace/projects/
├── assets/
│   ├── index.html          # 前端浏览页面
│   ├── admin.html          # 管理后台
│   ├── site_config.json    # 站点配置文件
│   └── guowen_huitong_data.json  # 资料数据
├── config/
│   └── agent_llm_config.json
├── CNAME                   # 域名配置
├── vercel.json             # Vercel 配置
└── netlify.toml            # Netlify 配置
```

### 关键特性

1. **登录保护**
   - 使用 sessionStorage 存储登录状态
   - 默认密码: admin888
   - 支持 ESC 键关闭模态框

2. **客服系统**
   - 悬浮式设计，固定在右下角
   - 支持多个客服配置
   - 二维码图片 URL 配置
   - 可独立控制每个客服的显示状态

3. **邀请码系统**
   - 可配置邀请链接
   - 可自定义按钮文字
   - 可控制显示位置

4. **响应式设计**
   - 图片灯箱预览
   - 视频自适应播放
   - 多设备适配

5. **动画效果**
   - CSS3 动画
   - Cubic-bezier 缓动函数
   - 硬件加速

## 📝 注意事项

1. **图片上传**
   - 客服二维码需要上传到图床或对象存储
   - 获取完整的图片 URL 并填入配置

2. **密码安全**
   - 生产环境请修改默认密码
   - 建议使用后端 API 验证

3. **配置保存**
   - 当前版本为前端演示模式
   - 配置仅保存在页面内存中
   - 刷新页面后配置会重置
   - 生产环境需要后端支持

4. **域名配置**
   - CNAME 文件已配置为 www.guowenhuitong.com
   - 需要在域名服务商处添加 DNS 记录
   - 记录类型: CNAME
   - 记录值: vercel.app 或 netlify.app（根据部署平台）
