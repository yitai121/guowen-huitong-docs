## 🎯 接下来要做什么？

只需要 5 个步骤，大约 5 分钟！

---

## 第一步：打开 Vercel 网站 🌐

**动作：**
1. 双击浏览器图标（推荐 Chrome）
2. 在地址栏输入：`https://vercel.com`
3. 按回车键

**你会看到什么：**
- 黑色背景的页面
- 顶部有 Vercel 的 logo
- 中间有 "Develop. Preview. Ship." 的文字
- 右上角有白色的 "Login" 按钮

**📍 提示：** 不要复制网址，手动输入更安全！

---

## 第二步：登录 Vercel 🔐

**动作：**
1. 点击右上角的白色 "Login" 按钮
2. 在弹出的页面中，点击黑色的 "Continue with GitHub" 按钮
3. 在 GitHub 授权页面，点击绿色的 "Authorize Vercel" 按钮

**你会看到什么：**
- 登录后会跳转到 Vercel 仪表板
- 左侧有导航菜单
- 中间显示项目列表
- 右上角显示你的头像

**📍 提示：** 如果是第一次使用，可能需要创建账号，输入用户名即可。

---

## 第三步：添加新项目 📦

**动作：**
1. 点击左上角的黑色 "Add New" 按钮（带有 + 加号图标）
2. 在弹出的下拉菜单中，点击 "Project"

**你会看到什么：**
- 页面标题：Import Git Repository
- 中间显示 GitHub、GitLab、Bitbucket 的图标
- GitHub 图标旁边有绿色的对勾 ✓
- 下面显示你的 GitHub 仓库列表

**📍 提示：** 如果 GitHub 图标没有绿色对勾，点击它进行连接。

---

## 第四步：导入仓库 📥

**动作：**
1. 向下滚动，找到 `Your GitHub Repositories` 部分
2. 找到这个仓库：`guowen-huitong-docs`
3. 点击右侧蓝色的 "Import" 按钮

**你会看到什么：**
- 页面显示项目配置信息
- Project Name: `guowen-huitong-docs`
- Framework: `Other` 或 `Static`
- Root Directory: `./`

**📍 提示：** 所有配置保持默认，不需要修改任何设置！

---

## 第五步：开始部署 🚀

**动作：**
1. 向下滚动到页面底部
2. 点击大的蓝色 "Deploy" 按钮
3. 等待 1-2 分钟

**你会看到什么：**
- 页面显示部署进度
- 状态从 "Queued" → "Building" → "Ready"
- 最后显示 "Congratulations!" 页面
- 右侧有一个蓝色的 "Visit" 按钮

**📍 提示：** 整个过程自动进行，你只需要等待！

---

## 第六步：访问你的网站 🎉

**动作：**
1. 点击蓝色的 "Visit" 按钮
2. 或手动输入：`https://guwen-huitong-docs.vercel.app`

**你会看到什么：**
- 全新的国文汇通网站
- 浮动的彩色球体背景（紫色、粉色、蓝色）
- 漂浮的白色小粒子
- 毛玻璃卡片效果
- 紫蓝粉渐变按钮

**📍 提示：** 尝试移动鼠标，观察粒子效果！

---

## 🆘 常见问题快速解决

### 问题 1：找不到 "Add New" 按钮
**解决：** 按 F5 刷新页面

### 问题 2：找不到 guowen-huitong-docs 仓库
**解决：**
- 访问 https://github.com/yitai121/guowen-huitong-docs
- 确认仓库存在
- 刷新 Vercel 页面

### 问题 3：部署失败
**解决：**
- 检查 GitHub 仓库是否有代码
- 点击 "Redeploy" 按钮重试

### 问题 4：网站无法访问
**解决：**
- 等待 1-2 分钟
- 清除浏览器缓存
- 使用无痕模式访问
