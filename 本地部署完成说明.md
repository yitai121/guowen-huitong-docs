## 🤖 自动化管理

### 使用管理脚本：

```bash
# 查看状态
./manage.sh status

# 启动服务器
./manage.sh start

# 停止服务器
./manage.sh stop

# 重启服务器
./manage.sh restart

# 查看日志
./manage.sh logs

# 显示帮助
./manage.sh help
```

## 🌍 让外网访问

### 方法 1：端口转发（推荐）

**如果小龙虾环境支持端口转发：**

在小龙虾控制台配置端口转发：
- 映射端口：8000
- 协议：HTTP
- 外网访问：会提供一个公网 IP 和端口

**访问地址示例：**
```
http://你的公网IP:外网端口
```

---

### 方法 2：使用内网穿透工具

**使用 ngrok 等工具（如果支持）：**

```bash
# 安装 ngrok（如果可用）
# 运行以下命令：
ngrok http 8000
```

这会生成一个临时的公网 URL。

---

### 方法 3：使用云服务器部署

**如果需要稳定的公网访问：**

1. 购买云服务器（阿里云、腾讯云等）
2. 上传代码到服务器
3. 配置 Nginx
4. 配置域名解析

## 🔧 故障排除

### 如果网站无法访问：

**1. 检查服务器状态：**
```bash
./manage.sh status
```

**2. 查看日志：**
```bash
./manage.sh logs
```

**3. 重启服务器：**
```bash
./manage.sh restart
```

**4. 检查端口是否被占用：**
```bash
netstat -tlnp | grep :8000
```
