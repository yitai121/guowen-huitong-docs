# 国文汇通资料管理系统 - 使用指南

## 📖 快速入门

### 第一步：打开系统
在浏览器中打开 `assets/index.html` 文件。

### 第二步：上传文档
1. 点击右上角的"上传文档"按钮
2. 填写文档信息：
   - 标题（必填）
   - 文件路径（必填，可以是本地路径或 URL）
   - 分类（必填）
   - 标签（可选，多个标签用逗号分隔）
   - 描述（可选）
3. 点击"上传"按钮

### 第三步：浏览文档
- 在搜索框输入关键词进行搜索
- 点击分类标签进行筛选
- 点击"搜索"按钮执行搜索
- 点击"重置"按钮清空搜索条件

### 第四步：下载文档
1. 在文档列表中找到目标文档
2. 点击"下载"按钮
3. 系统会生成一个 24 小时有效的下载链接

## 🎯 核心功能

### 1. 文档上传

**支持的文件类型：**
- PDF 文档
- Word 文档
- Excel 表格
- PowerPoint 演示文稿
- 文本文件
- 图片文件
- 压缩包

**上传方式：**
1. **本地文件上传**：填写本地文件路径
2. **URL 上传**：填写文件的 URL 地址

**注意事项：**
- 文件标题不能为空
- 分类必须选择或输入
- 标签可以有多个，用逗号分隔
- 描述可以帮助快速定位文档

### 2. 文档搜索

**搜索方式：**
- **关键词搜索**：在搜索框输入关键词
- **分类筛选**：点击分类标签
- **组合搜索**：同时使用关键词和分类

**搜索范围：**
- 文档标题
- 文档描述
- 文档标签

**技巧：**
- 使用多个关键词提高搜索精确度
- 选择分类缩小搜索范围
- 点击"重置"清空搜索条件

### 3. 文档下载

**下载流程：**
1. 在文档列表中找到目标文档
2. 点击"下载"按钮
3. 系统返回下载链接（24 小时有效）
4. 复制链接到浏览器下载

**注意事项：**
- 下载链接 24 小时后失效
- 每次下载都会增加下载次数统计
- 系统会自动记录下载行为

### 4. 数据统计

**统计信息：**
- 总文档数
- 总存储空间
- 总下载次数
- 分类统计

**自动更新：**
- 每 6 小时自动更新一次统计数据

### 5. 网络搜索

**功能说明：**
- 使用 AI 驱动的网络搜索
- 自动生成相关资料
- 自动添加到资料库

**使用方法：**
1. 点击"网络搜索"按钮
2. 填写搜索信息：
   - 关键词（必填）
   - 分类（必填）
   - 最大结果数（可选，默认 10）
3. 点击"搜索"按钮
4. 系统自动搜索并添加结果

## ⚙️ 自动化功能

### 定时任务
系统会自动执行以下任务：

1. **数据同步**（每小时）
   - 同步最新文档数据
   - 更新文档统计

2. **网络搜索**（每天 09:00）
   - 搜索相关资料
   - 自动添加到资料库

3. **统计更新**（每 6 小时）
   - 更新文档统计信息
   - 计算存储使用量

4. **数据备份**（每天 03:00）
   - 备份所有文档数据
   - 生成备份文件

5. **清理旧文件**（每周日 02:00）
   - 清理 30 天前的旧文件
   - 释放存储空间

6. **健康检查**（每 15 分钟）
   - 检查数据库连接
   - 检查存储服务状态

### 手动触发
可以通过 Agent 工具手动触发自动化任务：
```python
# 使用 heartbeat_workflow 工具
heartbeat_workflow()
```

## 🚨 常见问题

### Q1: 上传失败怎么办？
**A:** 检查以下几点：
- 文件路径是否正确
- 文件是否存在
- 网络连接是否正常
- 文件大小是否超过限制

### Q2: 搜索不到文档？
**A:** 确认以下几点：
- 关键词是否正确
- 分类选择是否正确
- 文档是否已上传成功
- 搜索索引是否已更新

### Q3: 下载链接失效？
**A:** 下载链接 24 小时后自动失效，需要重新生成。

### Q4: 存储空间不足？
**A:** 可以：
- 删除不需要的文档
- 清理旧文件
- 升级存储方案

### Q5: 自动化任务不执行？
**A:** 检查：
- 自动化脚本是否运行
- 系统时间是否正确
- 日志文件是否有错误
