## 🍎 一、本地部署（Mac M 芯片）

#### 1. 创建项目目录
```bash

cd ~
mkdir guowen-huitong-website
cd guowen-huitong-website
```

#### 2. 创建管理脚本
```bash
# 创建 manage.sh 文件（从 Mac_M芯片本地部署指南.md 复制）
```

#### 3. 添加执行权限
```bash
chmod +x manage.sh
```

## 🤖 二、集群管理

### 第一步：初始化集群

```bash
# 添加执行权限
chmod +x cluster.sh

# 初始化集群
./cluster.sh init
```

**输出示例：**
```
╔═══════════════════════════════════════════════════════════════╗
║           🤖 小龙虾集群管理系统                    ║
╚═══════════════════════════════════════════════════════════════╝

ℹ️  初始化集群...
✅ 集群目录已创建
ℹ️  集群目录: /Users/你的用户名/guowen-huitong-cluster
```

---

### 第二步：添加实例

#### 方法 1：使用交互式菜单
```bash
./cluster.sh
```

然后选择 `2) 添加新实例`

#### 方法 2：使用命令行
```bash
./cluster.sh add
```

**输入示例：**
```
实例名称: instance1
端口 (默认8001): 8001
描述 (可选): 主站
```

**输出示例：**
```
✅ 实例 'instance1' 已创建
ℹ️  端口: 8001
ℹ️  目录: /Users/用户名/guowen-huitong-cluster/instance1
```

---

### 第三步：批量启动所有实例

#### 方法 1：使用交互式菜单
```bash
./cluster.sh
```

## 📊 三、常用命令

### 单实例管理

```bash
# 启动单个实例
./cluster.sh start instance1

# 停止单个实例
./cluster.sh stop instance1

# 查看实例状态
./cluster.sh status
```

### 批量操作

```bash
# 批量启动所有实例
./cluster.sh start-all

# 批量停止所有实例
./cluster.sh stop-all

# 批量部署到所有实例
./cluster.sh deploy
```

### 实例管理

```bash
# 列出所有实例
./cluster.sh list

# 删除实例
./cluster.sh delete instance1

# 初始化集群
./cluster.sh init
```

### 交互式菜单

```bash
# 进入交互式菜单
./cluster.sh
```

**菜单选项：**
```
请选择操作：

  1) 初始化集群
  2) 添加新实例
  3) 列出所有实例
  4) 查看集群状态
  5) 批量启动所有实例
  6) 批量停止所有实例
  7) 批量部署
  8) 删除实例
  9) 退出
```

## 🔧 六、故障排除

### 问题 1：端口被占用

**错误信息：**
```
Address already in use
```

**解决方法：**
```bash
# 查找占用端口的进程
lsof -i :8001

# 杀死进程
kill -9 <PID>

# 重新启动
./cluster.sh start-all
```

---

### 问题 2：实例启动失败

**解决方法：**
```bash
# 查看实例日志
tail -20 ~/guowen-huitong-cluster/instance1/app.log

# 检查端口配置
cat ~/guowen-huitong-cluster/instance1/config.json

# 重启实例
./cluster.sh stop instance1
./cluster.sh start instance1
```

---

### 问题 3：无法访问实例

**检查清单：**

```bash
# 1. 检查实例状态
./cluster.sh status

# 2. 检查端口是否监听
lsof -i :8001

# 3. 检查防火墙
# macOS 防火墙设置 -> 系统偏好设置 -> 安全性与隐私 -> 防火墙
```
