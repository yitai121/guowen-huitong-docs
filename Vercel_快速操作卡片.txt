【5 步完成部署】

───────────────────────────────────────────────────────────────────

步骤 1️⃣: 打开 Vercel（1 分钟）

在浏览器地址栏输入：
https://vercel.com

按回车访问


───────────────────────────────────────────────────────────────────

步骤 2️⃣: 登录 Vercel（1 分钟）

1. 点击右上角白色按钮 "Login"
2. 点击 "Continue with GitHub"
3. 在 GitHub 授权页面点击绿色按钮 "Authorize Vercel"
4. 登录成功，进入 Vercel 仪表板


───────────────────────────────────────────────────────────────────

步骤 3️⃣: 导入仓库（1 分钟）

1. 点击左上角黑色按钮 "Add New"
2. 点击下拉菜单中的 "Project"
3. 确认 GitHub 已连接（有绿色对勾 ✓）
4. 找到并点击仓库: guowen-huitong-docs
5. 点击右侧蓝色按钮 "Import"


───────────────────────────────────────────────────────────────────

步骤 4️⃣: 配置并部署（1 分钟）

1. 检查配置信息：
   ✓ Project Name: guowen-huitong-docs
   ✓ Framework: Other
   ✓ Root Directory: ./

2. 点击底部蓝色按钮 "Deploy"

3. 等待 1-2 分钟，看到 "Congratulations!" 就成功了！


───────────────────────────────────────────────────────────────────

步骤 5️⃣: 访问你的网站（1 分钟）

点击右侧蓝色按钮 "Visit"

或手动输入：
https://guwen-huitong-docs.vercel.app

【如果遇到问题】

Q1: 找不到 "Add New" 按钮
   → 刷新页面（F5）
   → 确认已登录

Q2: 找不到 guowen-huitong-docs 仓库
   → 访问 https://github.com/yitai121/guowen-huitong-docs
   → 确认仓库存在后刷新 Vercel 页面

Q3: 部署失败
   → 检查 GitHub 仓库是否有代码
   → 点击 "Redeploy" 按钮重试

Q4: 网站无法访问
   → 等待 1-2 分钟（DNS 传播）
   → 清除浏览器缓存（Ctrl + Shift + Delete）
   → 使用无痕模式访问
