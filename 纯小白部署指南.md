## 🌐 第二步：部署到公网（让所有人都能访问）

### 方案一：使用 Vercel（最简单，推荐）

#### 准备工作（需要你做的）
1. 注册一个 GitHub 账号（如果没有）
   - 访问：https://github.com/signup
   - 按照提示注册

2. 注册一个 Vercel 账号（如果没有）
   - 访问：https://vercel.com/signup
   - 使用 GitHub 账号登录

#### 步骤 1：创建 GitHub 仓库

1. 登录 GitHub
2. 点击右上角的 `+` 号
3. 选择 `New repository`
4. 填写信息：
   - Repository name: 输入 `guowen-huitong-docs`
   - Public/Private: 选择 `Public`
   - 勾选 `Add a README file`（可选）
5. 点击绿色按钮 `Create repository`

#### 步骤 2：配置 Git（需要你操作）

**如果是在 Windows 电脑上：**

1. 下载并安装 Git
   - 访问：https://git-scm.com/download/win
   - 下载后双击安装
   - 全部点"下一步"即可

2. 打开 Git Bash
   - 在电脑开始菜单搜索 "Git Bash"
   - 点击打开

3. 配置用户信息（只需要第一次）
   ```bash
   git config --global user.name "你的名字"
   git config --global user.email "你的邮箱"
   ```

#### 步骤 3：推送代码到 GitHub

**在 Git Bash 中执行以下命令：**

```bash
# 1. 进入项目目录（如果你不在项目目录）
cd /path/to/workspace/projects

# 2. 初始化 Git 仓库
git init

# 3. 添加所有文件
git add .

# 4. 提交代码
git commit -m "feat: 全面UI重构 - 互联网大厂级视觉设计"

# 5. 添加 GitHub 仓库地址
# 注意：把下面的地址改成你自己的仓库地址
git remote add origin https://github.com/你的GitHub用户名/guowen-huitong-docs.git

# 6. 推送代码到 GitHub
git branch -M main
git push -u origin main
```

**如果提示输入用户名和密码：**
- 用户名：输入你的 GitHub 用户名
- 密码：使用 GitHub Personal Access Token（不是登录密码）
  - 如何获取 Token？
  1. GitHub -> Settings -> Developer settings -> Personal access tokens
  2. 点击 "Generate new token"
  3. 勾选 `repo` 权限
  4. 点击生成，复制 token

#### 步骤 4：在 Vercel 部署

1. 登录 Vercel：https://vercel.com
2. 点击页面上的 `Add New Project`
3. 找到你的 `guowen-huitong-docs` 仓库
4. 点击 `Import`
5. 点击 `Deploy` 按钮
6. 等待 1-2 分钟，看到 "Congratulations!" 就成功了

#### 步骤 5：访问你的网站

部署完成后，Vercel 会给你一个网址，比如：
```
https://guwen-huitong-docs.vercel.app
```

在浏览器打开这个网址，你就能看到新网站了！

## 🎨 第三步：绑定你的域名 www.guowenhuitong.com

### 在 Vercel 配置自定义域名

1. 登录 Vercel
2. 找到你的项目
3. 点击项目名称进入
4. 点击顶部的 `Settings` 标签
5. 点击左侧的 `Domains`
6. 在输入框中输入：`www.guowenhuitong.com`
7. 点击 `Add`

### 配置 DNS 记录

Vercel 会告诉你需要添加的 DNS 记录，通常有两种方式：

#### 方法 1：使用 Vercel DNS（推荐）

1. 在 Vercel 的 Domains 页面
2. 看到提示，点击 `Configure DNS`
3. 选择 `Use Vercel DNS`
4. 按照提示操作

#### 方法 2：在你的域名服务商配置

如果你在阿里云、腾讯云、GoDaddy 等购买了域名：

1. 登录你的域名服务商
2. 找到域名管理
3. 找到 DNS 解析设置
4. 添加记录：

**记录类型**：CNAME
**主机记录**：www
**记录值**：guwen-huitong-docs.vercel.app
**TTL**：600

5. 保存

### 等待生效

DNS 解析需要几分钟到几小时不等，通常 10-30 分钟生效。

## 🆘 遇到问题怎么办？

### 问题 1：localhost 打不开
**解决方法**：
1. 确认输入的是 `http://localhost:8000`（不是 https）
2. 检查本地服务器是否在运行
3. 尝试换一个端口，比如 `http://localhost:3000`

### 问题 2：GitHub 推送失败
**解决方法**：
1. 检查用户名和密码是否正确
2. 使用 Personal Access Token 而不是登录密码
3. 确保仓库地址正确

### 问题 3：Vercel 部署失败
**解决方法**：
1. 查看 Vercel 的日志
2. 检查代码是否有错误
3. 重新推送代码

### 问题 4：域名无法访问
**解决方法**：
1. 检查 DNS 记录是否配置正确
2. 等待 DNS 传播（最多 24 小时）
3. 使用在线 DNS 检查工具
