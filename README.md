# 国文汇通 - 企业知识管理平台

现代化、高大上的企业知识管理网站。

## ✨ 特性

- 🎨 现代化 UI 设计（动态渐变背景、粒子效果、玻璃态设计）
- 📱 响应式布局（适配手机和电脑）

## 🚀 快速开始

### 本地运行

```bash
# 使用 Python 启动本地服务器
python3 -m http.server 8000

# 访问
http://localhost:8000
```

### 部署到 Vercel

1. 推送代码到 GitHub
2. 在 Vercel 导入仓库
3. 点击 Deploy
