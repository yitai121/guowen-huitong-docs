# 国文汇通资料管理系统

一个现代化、高大上的资料管理系统，支持文件上传、搜索、下载、网络搜索、自动化更新等功能。

## ✨ 核心特性

### 🎨 现代化 UI 设计
- 高大上的渐变背景和动画效果
- 丝滑流畅的交互动画
- 响应式设计，支持多端访问
- 深色主题，护眼舒适

### 📁 文件管理
- 支持本地文件上传
- 支持从 URL 上传文件
- 自动识别文件类型和大小
- 支持添加标题、描述、分类、标签
- 文档列表展示和详情查看
- 文件下载功能

### 🔍 智能搜索
- 按关键词搜索（标题、描述、标签）
- 按分类筛选
- 实时搜索结果更新
- 支持模糊匹配

### 🌐 网络搜索
- AI 驱动的智能网络搜索
- 自动生成相关资料标题和描述
- 自动添加到资料库
- 支持自定义分类

### ⚙️ 自动化操作
- 自动数据同步（每小时）
- 自动网络搜索（每天）
- 自动统计数据更新（每6小时）
- 自动数据备份（每天）
- 自动清理旧文件（每周）
- 健康检查（每15分钟）

### 📊 数据统计
- 总文档数统计
- 存储空间使用情况
- 下载次数统计
- 分类统计
- 实时更新

## 🚀 快速开始

### 1. 访问系统
打开浏览器，访问 `assets/index.html`

### 2. 上传文档
1. 点击"上传文档"按钮
2. 填写文档标题、文件路径、分类、标签、描述
3. 点击"上传"完成

### 3. 搜索文档
1. 在搜索框输入关键词
2. 选择分类标签（可选）
3. 实时查看搜索结果

### 4. 下载文档
1. 在文档列表中找到目标文档
2. 点击下载按钮
3. 获取 24 小时有效的下载链接

### 5. 网络搜索
1. 点击"网络搜索"按钮
2. 输入搜索关键词和分类
3. 系统自动搜索并添加相关资料

## 🛠️ 技术架构

### 后端技术栈
- **Python** - 主要开发语言
- **Supabase** - 数据库（PostgreSQL）
- **S3 兼容对象存储** - 文件存储
- **LangChain** - Agent 框架
- **FastAPI** - Web 框架（后端 API）

### 前端技术栈
- **HTML5** - 页面结构
- **CSS3** - 样式设计
- **JavaScript** - 交互逻辑
- **原生实现** - 无需框架依赖

### 核心组件
```
/workspace/projects/
├── assets/
│   ├── index.html              # 前端界面
│   └── guowen_huitong_data.json # 国文汇通资料配置
├── config/
│   └── agent_llm_config.json   # Agent 配置
├── scripts/
│   ├── automation.py           # 自动化脚本
│   └── init_guowen_data.py     # 初始化脚本
├── src/
│   ├── agents/
│   │   └── agent.py            # Agent 核心代码
│   ├── tools/
│   │   └── document_manager.py # 资料管理工具
│   └── storage/
│       └── database/
│           ├── shared/
│           │   └── model.py     # 数据库模型
│           └── supabase_client.py # 数据库客户端
└── requirements.txt            # 依赖包
```
