# 🚀 3分钟自动化部署指南

## 📌 你需要准备的信息

### 必需信息（填入脚本即可）：
1. ✅ GitHub 用户名
2. ✅ GitHub 邮箱
3. ✅ GitHub Personal Access Token
4. ✅ 你的名字

---

## 🎯 Windows 用户（推荐）

### 步骤 1：获取 GitHub Token（2分钟）

1. 访问：https://github.com/settings/tokens
2. 点击 `Generate new token` → `Generate new token (classic)`
3. 填写信息：
   - Note: 输入 `guowen-deploy`
   - Expiration: 选择 `90 days` 或 `No expiration`
   - 勾选权限：`repo`（这会自动勾选所有子选项）
4. 点击底部绿色按钮 `Generate token`
5. **重要！** 复制生成的 token（只显示一次）

### 步骤 2：编辑部署脚本（1分钟）

1. 找到文件：`auto_deploy.bat`（就在当前目录）
2. 右键点击文件，选择 `编辑`（用记事本打开）
3. 填写第 7-13 行的配置信息：

```batch
set GITHUB_USERNAME=你的GitHub用户名
set GITHUB_REPO_NAME=guowen-huitong-docs
set GITHUB_EMAIL=你的GitHub邮箱
set GITHUB_TOKEN=刚才复制的token
set GIT_NAME=你的名字
```

**示例：**
```batch
set GITHUB_USERNAME=zhangsan
set GITHUB_REPO_NAME=guowen-huitong-docs
set GITHUB_EMAIL=zhangsan@example.com
set GITHUB_TOKEN=ghp_xxxxxxxxxxxxxxxxxxxxx
set GIT_NAME=张三
```

4. 保存文件（Ctrl+S），关闭记事本

### 步骤 3：运行脚本（30秒）

1. 双击 `auto_deploy.bat` 文件
2. 等待脚本自动执行（大约 1-2 分钟）
3. 看到 `🎉 自动部署第一步完成！` 就成功了

---

## 🐧 Linux/Mac 用户

### 步骤 1：获取 GitHub Token

同 Windows 用户的步骤 1

### 步骤 2：编辑部署脚本

1. 打开终端
2. 编辑脚本文件：
   ```bash
   nano auto_deploy.sh
   ```
3. 填写第 9-15 行的配置信息：

```bash
GITHUB_USERNAME="你的GitHub用户名"
GITHUB_REPO_NAME="guowen-huitong-docs"
GITHUB_EMAIL="你的GitHub邮箱"
GITHUB_TOKEN="刚才复制的token"
GIT_NAME="你的名字"
```

4. 保存文件：按 `Ctrl+O`，然后按 `Enter`
5. 退出编辑器：按 `Ctrl+X`

### 步骤 3：运行脚本

1. 添加执行权限：
   ```bash
   chmod +x auto_deploy.sh
   ```

2. 运行脚本：
   ```bash
   bash auto_deploy.sh
   ```

3. 等待脚本自动执行（大约 1-2 分钟）

## 🌐 步骤 4：在 Vercel 部署（3分钟）

### 1. 访问 Vercel
打开浏览器访问：https://vercel.com

### 2. 登录并导入项目
1. 用 GitHub 账号登录
2. 点击页面上的 `Add New Project`
3. 找到你的 `guowen-huitong-docs` 仓库
4. 点击 `Import` 按钮

### 3. 开始部署
1. 确认项目信息，点击 `Deploy` 按钮
2. 等待 1-2 分钟，看到 `Congratulations!` 就成功了

### 4. 访问你的网站
Vercel 会给你一个网址，比如：
```
https://guwen-huitong-docs.vercel.app
```

复制这个网址，在浏览器打开，你会看到全新的界面！

---

## 🌟 步骤 5：绑定自定义域名（可选）

如果你想使用 `www.guowenhuitong.com`：

### 1. 在 Vercel 添加域名
1. 打开 Vercel 项目
2. 点击 `Settings` 标签
3. 点击左侧 `Domains`
4. 输入：`www.guowenhuitong.com`
5. 点击 `Add`

### 2. 配置 DNS 记录
Vercel 会告诉你需要添加的 DNS 记录：

**使用 Vercel DNS（最简单）：**
1. 在 Vercel 的 Domains 页面
2. 点击 `Configure DNS`
3. 选择 `Use Vercel DNS`
4. 按照提示操作

**使用你自己的域名服务商：**
在你的 DNS 设置中添加：

| 记录类型 | 主机记录 | 记录值 |
|---------|---------|--------|
| CNAME   | www     | guwen-huitong-docs.vercel.app |

### 3. 等待 DNS 生效
通常需要 10-30 分钟，最多 24 小时
