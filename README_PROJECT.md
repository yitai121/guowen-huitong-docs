## 🚀 快速开始

### 本地开发
```bash
python -m http.server 8000 --directory assets
# 访问: http://localhost:8000/
```

### 直接打开
```
assets/index.html  # 浏览页面
assets/admin.html  # 管理后台
```

## 🚀 部署教程

### 推荐方案：Vercel ⭐⭐⭐⭐⭐

```bash
# 1. 安装 Vercel CLI
npm install -g vercel

# 2. 登录
vercel login

# 3. 部署
vercel

# 4. 生产环境
vercel --prod
```

**访问地址：**
- 浏览页面：`https://guowen-huitong-docs.vercel.app`
- 管理后台：`https://guowen-huitong-docs.vercel.app/admin`

详细部署教程请查看 [DEPLOY.md](./DEPLOY.md)
